Citizen Access Control System

Ensuring Fair Distribution of Subsidized Items in Malaysia

Final Year Project (2025)

Objectives

  • Implement a QR code–based verification system for subsidy eligibility.
  • Provide secure data storage using encryption mechanisms (Fernet AES).
  • Develop user and admin dashboards for subsidy management.
  • Evaluate QR scanning performance with GM65/GM67 barcode scanners.

Technologies & Tools

  • Python (Flask Framework)
  • MySQL Database (XAMPP)
  • QR Code Generator & Scanner (GM65 / GM67)
  • Encryption: Fernet (AES)
  • Grafana + Prometheus (Performance monitoring)
  • Raspberry Pi (System integration)

Methodology

The project followed a structured approach combining software engineering and system integration steps:

  1. Design system architecture for QR-based access control.
  2. Develop Flask-based web dashboards (User & Admin).
  3. Integrate GM65/GM67 scanners with Raspberry Pi.
  4. Implement encryption for user identity protection.
  5. Test & evaluate scanning performance using Grafana dashboards.

Impact

This project contributes to social equity by ensuring subsidies reach the intended recipients, while reducing financial leakage and misuse. It also aligns with Malaysia’s digital transformation initiatives (MyDigital & JENDELA) and minimizes environmental impact by eliminating paper-based verification.
