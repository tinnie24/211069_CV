Face Recognition for User Authentication (Engineering Algorithms)

This project, developed under the ECC3122 - Engineering Algorithms course, focused on building a real-time face recognition system for secure and efficient user authentication. It integrates machine learning models with algorithms such as Timsort and Linear Search to manage student records and authentication effectively.

üåÄ Timsort (Sorting)
Timsort is a hybrid algorithm that combines merge sort and insertion sort, with a time complexity of O(n log n). It was used to sort the list of student names both alphabetically and by detection order, ensuring quick lookups and efficient record management.

üîç Linear Search (Searching)
Linear search was used to check whether a student's name exists in the dataset or in the detected list. With a time complexity of O(n), it is efficient enough for our small dataset of 24 students, making it simple and effective for real-time use.
